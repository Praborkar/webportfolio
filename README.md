## 🚀 Cinematic Features

- **System Initialization**: A high-tech boot sequence featuring binary rainfall and volumetric branding.
- **Deep Space Engine**: A multi-layered parallax starfield with drifting nebulas and constellation geometric grids.
- **Tactical Command Hub**: 3D holographic project cards with inertia-based tilt and interactive terminal logs.
- **Tech-Singularity (3D Skills)**: An interactive 3D particle sphere that allows users to orbit through my tech stack.
- **Orbital Command Nav**: A floating glassmorphic pill-nav with scanning light-beam highlights and digital metadata markers.
- **Quantum Hero**: High-contrast "starlight" typography with 2x2 capability nodes and real-time status pulses.

## 🧪 Installation & Orbital Entry

To run this command center locally, follow these technical protocols:

### 1. Initialize Connection
```bash
git clone https://github.com/Praborkar/webportfolio.git
cd webportfolio
```

### 2. Configure Life Support (Environment Variables)
Create a `.env` file in the root directory:
```env
VITE_GITHUB_TOKEN=your_personal_access_token
VITE_GITHUB_USERNAME=Praborkar
VITE_EMAILJS_SERVICE_ID=your_id
VITE_EMAILJS_TEMPLATE_ID=your_id
VITE_EMAILJS_PUBLIC_KEY=your_key
```

### 3. Ignite Engines
```bash
npm install
npm run dev
```

---

## 🛰️ Deployment (Vercel)

The project includes a specialized `vercel.json` for SPA routing and security. When deploying:
1. Import the repo to [Vercel](https://vercel.com).
2. Add your `VITE_GITHUB_TOKEN` in the **Environment Variables** section.
3. Vercel will automatically build the `dist` folder via `npm run build`.
